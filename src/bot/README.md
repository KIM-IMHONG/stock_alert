# 🤖 한국 주식 알림 텔레그램봇

실시간으로 한국 주식의 급등/급락을 모니터링하고 텔레그램으로 알림을 보내는 봇입니다.

## 🚀 주요 기능

### 📋 기본 명령어
- `/start` - 봇 시작 및 사용자 등록
- `/add [종목코드]` - 모니터링 종목 추가 (최대 10개)
- `/remove [종목코드]` - 종목 제거
- `/list` - 내 종목 목록 조회
- `/settings` - 알림 조건 설정 (급등/급락 %)

### 💡 사용 예시
```
/add 005930    # 삼성전자 추가
/remove 000660 # SK하이닉스 제거
/settings 7    # 7% 이상 변동 시 알림 설정
```

### 🎯 핵심 기능
- **사용자별 종목 관리**: 각 사용자는 최대 10개 종목 모니터링 가능
- **알림 임계값 설정**: 1~50% 범위에서 급등/급락 기준 설정
- **중복 알림 방지**: 15분 쿨다운으로 스팸 방지
- **직관적인 UI**: 인라인 키보드로 쉬운 조작

## 🛠️ 설치 및 실행

### 1. 전제조건
- Python 3.8+
- 텔레그램 봇 토큰 ([@BotFather](https://t.me/botfather)에서 생성)

### 2. 설치
```bash
# 의존성 설치
pip install -r requirements.txt

# 환경 설정
cp .env.example .env
# .env 파일에 봇 토큰 설정
```

### 3. 실행
```bash
# 스크립트로 실행
./run_bot.sh

# 또는 직접 실행
python3 main.py
```

## ⚙️ 설정

### 환경 변수 (.env)
```env
TELEGRAM_BOT_TOKEN=your_bot_token_here
```

### 주요 설정값 (config.py)
- `MAX_WATCHLIST_SIZE = 10` - 최대 모니터링 종목 수
- `ALERT_COOLDOWN_MINUTES = 15` - 중복 알림 방지 시간
- `DEFAULT_ALERT_THRESHOLD = 5.0` - 기본 알림 임계값

## 📊 데이터베이스 구조

### Users 테이블
- `user_id` - 텔레그램 사용자 ID
- `username` - 텔레그램 사용자명
- `first_name` - 사용자 이름
- `alert_threshold` - 알림 임계값
- `created_at`, `updated_at` - 생성/수정 시간

### Watchlist 테이블
- `user_id` - 사용자 ID
- `stock_code` - 6자리 종목코드
- `added_at` - 추가 시간

### Alert History 테이블
- `user_id` - 사용자 ID
- `stock_code` - 종목코드
- `alert_type` - 알림 유형 (상승/하락)
- `price` - 당시 주가
- `change_rate` - 변동률
- `sent_at` - 발송 시간

## 🔧 아키텍처

### 핵심 클래스
1. **BotHandler** - 텔레그램 명령어 처리
2. **UserManager** - 사용자 및 종목 관리
3. **AlertSender** - 알림 발송 및 쿨다운 관리
4. **DatabaseManager** - 데이터베이스 연동

### 파일 구조
```
src/bot/
├── main.py              # 메인 실행 파일
├── config.py            # 설정 및 메시지 템플릿
├── requirements.txt     # 의존성
├── handlers/
│   └── bot_handler.py   # 봇 핸들러
├── managers/
│   ├── user_manager.py  # 사용자 관리
│   └── alert_sender.py  # 알림 발송
├── models/
│   └── database.py      # 데이터베이스 모델
└── utils/
    └── stock_utils.py   # 주식 관련 유틸
```

## 🚨 알림 시스템

### 알림 조건
- 설정한 임계값 이상 급등/급락 발생
- 15분 쿨다운 내 중복 알림 없음
- 사용자별 개별 임계값 적용

### 알림 메시지 예시
```
🚀 급등 알림!

📈 종목: 005930 삼성전자
💰 현재가: 75,000원
📊 변동률: +5.20%
⏰ 시간: 14:30:15

📈 5.20% 상승했습니다!
```

## 📝 로깅

- 파일 로그: `bot.log`

## 🔒 보안 고려사항

- 봇 토큰은 환경 변수로 관리
- SQLite 데이터베이스는 로컬 저장
- 사용자 데이터 최소화 수집
- 에러 핸들링으로 서비스 안정성 보장

## 🐛 문제 해결

### 일반적인 문제
1. **봇이 응답하지 않음**
   - 봇 토큰 확인
   - 네트워크 연결 확인
   - 로그 파일 확인

2. **데이터베이스 오류**
   - 파일 권한 확인
   - 디스크 용량 확인

3. **알림이 오지 않음**
   - 종목이 watchlist에 있는지 확인
   - 알림 임계값 설정 확인
   - 쿨다운 시간 확인
